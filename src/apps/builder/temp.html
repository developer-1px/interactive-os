서비스 특징

클라우드를 기반으로 한 실시간 이미지 변환

간편한 이미지 변환 환경 설정

Image Optimizer을 사용하려면 네이버 클라우드 플랫폼에서 제공하는 Object Storage, CDN, Cloud Log Analytics와 연동이 필요합니다. 네이버 클라우드 플랫폼 콘솔에서 간편하게 서비스 간 연동은 물론 변환 규칙을 설정하여 이미지 변환 환경을 설정할 수 있습니다.

사용자 설정 규칙 외 작업 방지

Image Optimizer은 사용자가 사전에 설정한 변환 규칙이 적용된 이미지 변환만 허용되기 때문에 의도하지 않았거나 악의적으로 이미지를 대량 변환하는 작업을 방지할 수 있습니다.

서비스 환경에 맞춘 요금 계산

이미지 변환 서비스인 Image Optimizer과 콘텐츠 전송 서비스인 CDN의 사용량이 구분되어 각각 요금이 산정됩니다. 서비스에 따라 이미지 변환이 많거나 CDN 배포가 많을 경우 사용량을 구분하여 과금하므로 불필요한 요금 지출을 줄일 수 있습니다.

검증된 이미지 변환 및 배포

Image Optimizer은 네이버가 오랜 시간 사용해 온 이미지 변환 엔진 및 서비스 아키텍처를 기반으로 노하우가 적용된 고품질의 안정적인 이미지 변환 서비스 제공을 보장합니다.

상세 기능

이미지 변환

원본 이미지를 서비스 용도에 맞추어 실시간으로 변환할 수 있습니다. 현재 제공하는 이미지 변환 기능은 아래와 같습니다. (변환 기능은 계속해서 추가될 예정입니다.)

크기 변환

크기 조정(resize), 잘라내기(crop), 가로/세로 기준 변환, 강제 변환 등 다양한 기준으로 이미지 크기를 조절할 수 있습니다.

워터마크 적용

출처나 저작권을 밝히는 데 사용되는 워터마크를 선택하고 위치를 조절하여 섬네일에 적용할 수 있습니다.

필터 효과 적용

  • Sharpen: 섬네일 이미지를 원본보다 선명하게 보여주는 효과입니다. 글자 이미지이거나 압축률이 높아 이미지가 흐려지는 경우에 사용할 수 있습니다.

  • Blur: 섬네일 이미지를 흐릿하게 만드는 효과를 적용할 수 있습니다.

  • Black & White: 섬네일 색상을 흑백으로 변경할 수 있습니다.

포맷 변환

변환할 타깃 섬네일의 파일 형식을 설정할 수 있습니다. ※ [입력 파일 형식] *.jpg, *.gif, *.png → [출력 파일 형식] *.jpg, *.gif, *.png

애니메이션 변환

원본이 *.gif 확장자(애니메이션)일 경우 섬네일을 동적 이미지로 변환할지 정적 이미지(첫 장 기준)로 변환할지 선택할 수 있습니다.

변환 규칙 설정 및 관리

다양한 이미지 변환 규칙을 설정하고 규칙별로 적용될 옵션을 미리 설정할 수 있습니다. 변환 규칙은 쿼리 스트링 형태로 작성할 수 있으며 작성된 규칙은 파일 형태로 내보거나 가져올 수 있어서 편리하게 다시 사용할 수 있습니다. 또한, 설정해 둔 변환 규칙에 따른 이미지 변환 작업만 허용되므로 의도하지 않은 변환 작업과 비용 발생을 방지할 수 있습니다.

상세 변환 이력 조회

로그 분석 서비스인 Cloud Log Analytics 서비스와 연동하여 Image Optimizer 서비스를 통해 실제 변환 작업을 수행한 일별 이력을 조회할 수 있어 서비스 운영을 상세하게 관리할 수 있습니다. 일별 이력은 Image Optimizer 메뉴에서도 조회할 수 있으며 더 자세한 로그는 Cloud Log Analytics 메뉴에서 확인할 수 있습니다.

섬네일 변환 미리보기

원본 이미지에 고객이 지정한 변환 규칙에 따라 만들어진 섬네일을 웹 콘솔에서 바로 확인할 수 있어 원하는 최적의 변환 규칙을 설정하기에 용이합니다.

임시 섬네일 암호화 기능

Image Optimizer 내부에 저장되는 임시 저장 파일을 SSE 암호화 방식을 이용해 보관할 수 있습니다. 이를 통해 안전한 서비스 운영이 가능합니다.

서비스 연동을 통한 효율성 제고

네이버 클라우드 플랫폼에서 제공하는 서비스와 연동하고 콘솔에서 관리함으로써 업무 효율을 높일 수 있습니다. 

  • Object Storage: 원본 이미지 저장, 이미지를 섬네일로 변환 및 배포

  • CDN: 안정적인 콘텐츠 송출

  • Cloud Log Analytics: 상세 로그 조회 

이밖에 서비스 단위인 각 Project에 연동된 Object Storage의 버킷, CDN 설정 등을 사용자가 원하는 용도에 따라 변경할 수도 있습니다.

      (VAT 별도)

      • 서비스 구축을 위한 개발 및 테스트 용도로 월 1,000건씩 무료 이용량이 제공됩니다.

      • 변환된 섬네일은 임시 저장소에 잠시 보관됩니다. 만약 변환 요청받은 섬네일이 임시 저장소에 보관되어 있을 경우, 원본 이미지를 다시 변환하지 않고 임시 저장소에서 대응합니다.

      • 임시 저장소에서 대응할 경우, 과금 요소인 변환 횟수에서 제외되므로, 실제 변환 요청 횟수보다 이용 요금이 적게 나올 수 있습니다. (※ 다만, 임시보관소 특성상, 섬네일 보관 주기를 보장하기는 어려운 점 참고하시기 바랍니다.)

      • 변환 횟수는 1,000건 단위로 올림 하여 계산됩니다.

      • 이용량은 월 단위로 갱신되며, 이월되지 않습니다.

      • 연동되는 서비스인 Object Storage, CDN, CLA는 별도의 이용 요금이 부과되며, 각 이용 요금은 서비스별 요금제를 참고하시기 바랍니다.

      요금 예시

      • Case 1) 당월 이미지 변환 1,600건 이용 시,

      ﻿2,000 원﻿ = 기본 무료구간 1,000건을 제외하고 남은 600건을 1,000건 단위로 올림 하여 계산

      • Case 2) 당월 이미지 변환 30,000건 이용 시,

      ﻿48,000 원﻿ = 1,000건/1,000건 x 0원 + (10,000건 - 1,000건)/1,000건 x 2,000 원﻿ + (30,000건 – 10,000건)/1,000건 x 1,500 원﻿﻿
