## 1. 개요

글 전체(L1-359)에서 **LLM이 쓴 티가 나는 표현**을 3가지 유형으로 분류하여 감별.

| 유형 | 설명 | 아이콘 |
|------|------|--------|
| **단정적 선언** | "~뿐입니다", "~만이 유일한", "반드시 ~이어야만" — 저자의 실용적 판단을 존재론적 선언으로 격상 | 🔴 |
| **과잉 수사** | "피눈물", "자비 없는", "차가운 도구들", "서늘하게" — 불필요하게 감정을 쥐어짜는 표현 | 🟡 |
| **인공적 대칭/나열** | 깔끔하게 떨어지는 대구, 3연발 선언 등 — 사람이 안 쓰는 패턴으로 인공적 느낌 | 🔵 |

## 4. 특이사항 — 1부는 깨끗하다

**프롤로그~1부(L1-105)에서는 LLM 톤 마커가 거의 없습니다.** 사용자가 직접 퇴고한 결과로 보입니다. 문제 구간은 **2부 후반(L130 이후)부터 3부~끝으로 갈수록 밀도가 높아집니다.** 특히 3부(L229-296)가 가장 심합니다 — 하네스/오라클 논증 구간에서 단정+수사가 동시에 폭발합니다.
