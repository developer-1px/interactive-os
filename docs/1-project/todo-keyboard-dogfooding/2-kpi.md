## 성공 기준

| 지표 | 현재 값 | 목표 값 | 측정 방법 |
|------|---------|---------|-----------|
| Dogfooding 시나리오 통과율 | 미측정 | 5/5 (100%) | PRD의 5개 시나리오를 키보드만으로 수행 |
| 키보드로 불가능한 핵심 작업 | 미측정 | 0개 | 마찰 발견 리스트 → 모두 해소 |
| E2E 테스트 (Playwright) | 0개 | ≥ 15개 | 키보드 시나리오 기반 E2E spec |
| Unit 테스트 유지 | 153개 | ≥ 153개 | `vitest run` — 기존 테스트 깨지지 않아야 함 |
| 포커스 복원 정확도 | 미측정 | 100% | 삭제/붙여넣기/Undo 후 포커스 위치 검증 |
