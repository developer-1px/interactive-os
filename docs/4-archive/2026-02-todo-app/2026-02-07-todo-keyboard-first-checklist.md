## 2. 체크리스트 (키보드 퍼스트 우선순위)

### Phase 1: Core Completeness (Quick Wins) — 예상 0.5~1일

- [ ] **Undo/Redo 키바인딩 등록**: `todoKeys.ts`의 global에 `⌘Z` → `UndoCommand`, `⌘⇧Z` → `RedoCommand` 추가
- [ ] **전체 선택 (`⌘A`)**: listView zone에 `⌘A` → OS `SELECT_ALL` 연동 (Zone `select.mode: 'multiple'` 필요)
- [ ] **History Middleware 검증**: `historyMiddleware.ts`가 실제 스냅샷을 올바르게 추적하고 있는지 E2E 검증

### Phase 2: Multi-Select — 예상 1~2일

- [ ] **Zone select mode 설정**: ListView Zone에 `select.mode: 'multiple'` 또는 `'toggle'` 적용
- [ ] **범위 선택 (`⇧↑`/`⇧↓`)**: ZIFT의 range select 동작 확인 및 연결
- [ ] **벌크 삭제**: 다중 선택 상태에서 `Backspace` → 선택된 모든 항목 삭제
- [ ] **벌크 복사/이동**: 다중 선택 + `⌘C`/`⌘X`/`⌘V` 동작

### Phase 3: Board View Enhancement — 예상 1~2일

- [ ] **카드 컬럼 간 이동**: `⌘←`/`⌘→` 로 포커스된 카드를 인접 컬럼으로 이동
- [ ] **2D 네비게이션 검증**: Board의 Nested Zone (column→cards) 방향키 동작 검증
- [ ] **Board View 편집 모드**: Board에서의 인라인 편집 키맵 (`Enter` → 편집, `Escape` → 취소)

### Phase 4: Power User (Overlay 패턴) — 예상 3~5일

- [ ] **컨텍스트 메뉴**: `⇧F10` → 포커스된 아이템에 대한 액션 메뉴 (삭제/편집/이동/복제)
- [ ] **검색/필터**: `⌘F` 또는 `/` → 실시간 검색 Overlay + 결과 Zone 네비게이션
- [ ] **커맨드 팔레트**: `⌘K` → 퍼지 검색으로 모든 커맨드 접근 (OS Overlay 시스템 필요)

### Phase 5: Pro (아키텍처 확장 필요) — 예상 5~7일

- [ ] **서브태스크**: `Tab` → 들여쓰기, `⇧Tab` → 내어쓰기 (Nested Zone 재귀 구조)
- [ ] **날짜 선택기**: `⌘⇧D` → Popover Zone + Calendar 위젯
- [ ] **태그/라벨**: `#` 트리거 → 인라인 자동완성
- [ ] **정렬 변경**: `⌘⇧S` → 정렬 기준 토글 (Toolbar Trigger)
